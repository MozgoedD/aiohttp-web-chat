## Simple chat on websockets

Based on AIOHTTP framework for Python 3.5.3+

## Installation

My version of Python is 3.8.0

Install all requirements to your virtualenv

```    
$ python3 -m venv env
$ source env/bin/activate
$ pip install -r requirements.txt
```

First you need to create a PostgreSQL database. Enter database information into the config.py file.

To initialize database run init_db.py
```
$ python3 init_db.py
 ```

    
To run this code just type to console

```
$ python3 main.py
 ```
    
Open browser

    http://0.0.0.0:8080
